Overview

This is a data visualization project, which is going to analyze 14 countries from G20 have high education spending, healthcare spending and military spending from 2011 to 2018. The datasets are download from WORLDBANK and cleaned with excel. Here is shared link connect to google sheets. The Visualization tool I use is GOOGLE VISUALIZATION API, and the website is building with the free template from TEMPLATED. The project is published on GitHub and Zenodo.
